# News-Event-Extraction
新闻事件抽取相关论文公开资料

**1、揭示更大的图景：通过多样化新闻检索实现全面的事件理解**

**作者**：益宣汤；园园石；孙益群；安东尼·金和东

**论文摘要**：获取不同的视角对于理解真实世界的事件至关重要，然而大多数新闻检索系统优先考虑文本相关性，导致冗余的结果和有限的视角暴露。我们提出了NewScope，这是一个用于多样化新闻检索的两阶段框架，它通过在句子级别上对语义变化进行显式建模来增强事件覆盖。第一阶段使用密集检索来检索主题相关内容，而第二阶段将句子级聚类和多样性感知重排序应用于表面互补信息。为了评估检索多样性，我们引入了三个可解释的度量，即平均成对距离、正聚类覆盖率和信息密度比，并构建了两个段落级基准：LocalNews和DSGlobal。实验表明，NewScope始终优于强基线，在不影响相关性的情况下实现了显著更高的多样性。我们的结果证明了细粒度、可解释建模在减少冗余和促进全面事件理解方面的有效性。数据和代码可在此HTTPS URL中找到。
